Как установить
===============
Для установки шаблона необходимо выполнить следующие действия:

1. Активировать PDA-версию в ПУ сайта (Общие настройки - Активизировать шаблоны для PDA версии сайта)
2. Сделайте бекап ваших шаблонов (http://адрес_вашего_сайта//panel/?a=backupt)
3. Загрузить в корень сайта папку pda_images с ее содержимым.
4. Вставить содержимое файла pda.css из архива в Таблицу стилей (CSS) (http://адрес_вашего_сайта/panel/?a=tmpl;m=3;t=3;pda=1) (Обязательно выберите в вариантах "PDA шаблон")
5. Вставить код шаблонов из файлов в папке tpl в соотвествии с названиями файлов. В случае отсутсвия файла с названием для конкретного модуля
	использовать аналогичный от модуля "Каталог статей" (например в шаблоне "страница материала и комментариев к нему" для модуля Доска объявлений нужно использовать код из файла "Каталог статей - страница материала и комментариев к нему" )
